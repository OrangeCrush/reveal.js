Process of the Web

Client-Server Architecture

  1. Client makes a HTTP request to a web server.

  2. Server gets request, runs necessary instructions.

  3. Server returns output of instructions to Client.

SQL

Structured Query Language


CREATE TABLE BOOKS(
   title     VARCHAR(256),
   author    VARCHAR(256),
   available BIT,
   ISBN      INT UNSIGNED NOT NULL PRIMARY KEY AUTO_INCREMENT
);

Tables are used to represent logical entities in a database.

Example Queries


INSERT INTO BOOKS 
Values('Harry Potter and the Deathly Hallows',
       'J.K. Rowling', '1', 0);
               

SELECT ISBN, title from BOOKS
WHERE author = 'J.K. Rowling'
AND available = '1';
               

UPDATE BOOKS
SET available='0'
WHERE ISBN=2314521;
               

DELETE FROM BOOKS
WHERE title like '%Twilight%';
               

DROP TABLE BOOKS;
